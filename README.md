<h2>[2023 한아전 팀 프로젝트] Project:Spector</h2>
세계 각지에서 발생하는 의문의 무력 분쟁과 테러의 배후에는 민간군사조직 '그레이 하운드(Greyhound)'가 있었다. <br>
막대한 자금력과 최첨단 무기로 국제 사회의 질서를 뒤흔드는 그들은 비밀 계획을 은밀히 진행하고 있다.<br>

그에 대한 첩보를 입수한 끝에, 적의 기지로 특수 정보기관 소속의 정예 요원, Speactor가 투입되는데...<br>

**모든 임무는 선택의 연속이다. 당신의 판단이 작전의 성패를 결정하고, 세계의 운명을 바꿀것이다.**
</div>

## 개요

- 프로젝트 이름: Project:Spector
- 프로젝트 기간: 2023.04 ~ 2023.11
- 개발 엔진 및 언어: Unreal 5.1, Visual Studio 2022, C++
- 멤버: 김영웅, 김동훈, 김학준, 이 현, 장정혁, 홍승언, 김찬빈, 이준혁, 진광렬, 채준영, 최우원
- 장르: 3인칭 스텔스 슈팅 게임

용량과 에셋 저작권 문제로 인하여 Content 폴더를 제외하였으므로 해당 프로젝트로는 정상적인 실행이 불가능합니다.
